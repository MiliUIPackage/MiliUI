
This addon contains the button flashing functionality for SpellFlash.

This addon was created to separate the spell flashing functionality from SpellFlash
so that it may be used as an entirely separate resource if desired.

SpellFlashCore API:
http://wow.curseforge.com/addons/spellflashcore/pages/api/
